# ap2-generated

Generated/shared gRPC contract package for AP2.

Contents:
- `payment/` - payment service request/response types and gRPC client/server bindings
- `order/` - order streaming request/response types and gRPC client/server bindings
- `codec/jsoncodec/` - JSON codec registration used by the services
